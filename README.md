# Student Management System

A robust, multi-role web application designed to streamline academic operations, track student performance, and automate administrative workflows. This system simplifies student enrollment, course management, grading, and attendance tracking through an intuitive interface.

## 🚀 Features

### 👤 Administrative Dashboard
* **User Management:** Create, update, and manage profiles for students, faculty, and staff.
* **Course & Class Scheduling:** Allocate classrooms, assign professors, and set timetables.
* **Analytics Overview:** High-level metrics showing total enrollment, attendance averages, and fee statuses.

### 👨‍🏫 Faculty Portal
* **Grade Management:** Input and update assignment, quiz, and final exam marks.
* **Attendance Tracking:** Mark daily or lecture-wise attendance with automated low-attendance warnings.
* **Student Performance Analytics:** View class averages and identify students requiring academic assistance.

### 🎓 Student Portal
* **Academic Dashboard:** View current GPA, registered courses, and class schedules.
* **Progress Tracking:** Access detailed grade reports and real-time attendance percentages.
* **Profile Management:** Update personal contact details and view fee invoices.
